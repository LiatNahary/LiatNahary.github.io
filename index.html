About Me

Hello,
My name is Liat and I am currently transitioning from the world of architecture to the world of engineering. How did I get from CAD to code you ask? I always loved designing and creating. I went to architecture school because I loved the idea of making something that affects our everyday life.

Unexpectedly, while I love the field of architecture I simply don’t want to be an architect. So why code? In my search to find what I DO want to do I took a coding course, and another one, then I wrote a project and I just can’t stop.

There is a lot to learn in programming, from the coding language to all the supporting technologies. I am interested in taking an idea and turning it into reality. I cannot explain the joy of experiencing the program I wrote working: Making my computer send a SMS to my phone, making words and commands turn into a playable game, or seeing my program run from the cloud for the first time. Programming lets me create something functional, solve problems and be creative.

Although now days I spend most of my time sharpening my programming skills, I do have plenty of other skills and hobbies. Here are three fun facts about me:
- I have over forty plants
- I speak three languages fluently (Hebrew, English and Spanish)
- I attended the same high school as Jim from The Office (I do not know him, unfortunately)

Work

I am proud to share a collection of my work. I am always working on something, this page will update as new projects are created. For more information on my work, visit my main GitHub page.

  • My Math Assistant

    This android application designed to aid beginner mathematicians in performing addition and subtraction. The app motivates users to solve math problems by providing a test feature and a button counter, while also keeping track of tens and units transferred. Developed using Java for the back end in Android Studio, the app features original UI design.

  • Automatic Personalized Birthday Wish Sender

    This application runs automatically, generating and sending personalized birthday wishes to contacts from a database on the day of their birthday. Written in Java on Spring Boot framework using Rest API, Postman, Gradle, Twilio API, AWS and more.

  • Card Game: War

    This application is an implementation of the classic card game. Written in Java on Spring Boot framework, with Gradle, using Java Swing for UI.

  • Weekly Menu Generator

    This Application creates a unique weekly menu from recipe files. Written in Java on Spring Boot framework using Rest API, Postman and Gradle.
